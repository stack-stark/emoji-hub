# Emoji Hub

这是一个使用 umi4 构建的 emoji 网站，通过分类展示各种 emoji ，支持搜索功能。配合 vscode 插件快捷 emoji，可以快速查看以及使用 emoji 。

## 关于快捷 emoji 🐉

这是一个在编辑区输入`000`+ emoji 名字（中文）就可以生成 emoji 的 vs code 插件，内置三千多个 emoji，可以让我们快速生成 emoji！
